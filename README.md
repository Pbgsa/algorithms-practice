# Algorithms Practice

This repository contains implementations of classic algorithms and data structures used in technical interview preparation.

## Topics Covered

### Data Structures
- Arrays
- Linked Lists
- Stacks
- Queues
- Hash Tables

### Graph Algorithms
- Breadth First Search (BFS)
- Depth First Search (DFS)
- Dijkstra
- Union Find

### Sorting
- Merge Sort
- Quick Sort

### Problem Solving
Solutions to algorithmic problems from competitive programming and interview preparation.

## Languages

C++  
Python

## Goal

The goal of this repository is to practice problem solving and improve algorithmic thinking for software engineering interviews.
